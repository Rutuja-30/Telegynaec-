Telegynaec is a telemedicine-based mobile health platform designed to address delayed diagnosis and limited gynecological access among rural women, particularly those affected by PCOS/PCOD. The system integrates symptom tracking, machine learning-based risk assessment, and remote consultation workflows to support early screening and personalized care.

The project involved building an end-to-end telehealth and ML workflow, including:

• Collaborating with a pathology laboratory to obtain clinically validated PCOS/PCOD datasets
• Cleaning, preprocessing, and structuring laboratory and clinical parameters for supervised learning
• Implementing and evaluating a Random Forest classifier for PCOS risk prediction
• Performing feature selection and model evaluation using classification metrics (accuracy, precision, recall, F1-score)
• Developing backend architecture using Django and PostgreSQL for secure data storage and user management
• Integrating teleconsultation workflows, menstrual cycle, and symptom tracking to improve follow-up adherence.
